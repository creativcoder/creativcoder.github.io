Open source projects and contributions.

## Projects:

• [Avrow](https://github.com/creativcoder/avrow) - Avrow is a pure Rust implementation of the Avro specification with serde support.

• [Cargo-docco](https://github.com/creativcoder/cargo-docco) - Cargo subcommand to generate literate programming style source code documentation. 

• [Templar](https://github.com/creativcoder/templar): A simple template engine in Python

• [Brainfcuk](https://github.com/creativcoder/brainfcuk) - An optimizing brainfuck interpreter written in Rust using a custom intermediate representation.

• [Fender](https://github.com/creativcoder/fender) - A web scraper implemented using Jon's fantoccini library.
